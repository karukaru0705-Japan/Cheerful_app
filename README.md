# チアフル会計アプリ

ママさんバレー「チアフル」の会計を、スマホで入力・集計・Excel書き出しできる Web アプリ（PWA）です。

## 特長
- スマホのブラウザで動作。インストール不要（「ホーム画面に追加」でアプリ風に使えます）
- **オフライン対応**（電波がなくても入力・集計OK）
- データは**端末内に保存**（IndexedDB）。外部に送信しません
- **集計期間を自分で設定**（年度・四半期・半期・カスタム月数／開始年月の指定、または手動で開始〜終了）
- 提出フォーム（収入／支出／合計）を **Excel(.xlsx)** で書き出し
- 帳簿残高と実際の現金残高の**整合チェック**（今回の「500円不足」のようなズレを検出）

## 集計期間について
「設定」タブの「集計期間」で、集計の対象期間を決められます。
- **スケール**：単位（四半期=3ヶ月／半期=6ヶ月／年度=12ヶ月／カスタム月数）＋開始年月 → 期間が自動で決定
- **手動**：開始年月・終了年月を自由に指定

設定した期間内の取引だけが「一覧」「集計」「Excel」に反映されます。**期間外の取引は隠れるだけで消えません**（期間を切り替えれば再表示）。期首残高（前年度繰越金）は期間ごとに手動で設定してください。

## 画面
| タブ | できること |
|---|---|
| 入力 | 日付・収入/支出・カテゴリ・金額・摘要・メモ・領収書写真を入力（部費を除く） |
| 一覧 | 取引の一覧と残高の自動計算。編集・削除、写真の拡大 |
| 部費 | メンバー登録と、メンバー×月のグリッドで納付状況を管理（後述） |
| 集計 | カテゴリ別集計、収入合計／支出合計／差引残高、整合チェック、会計報告プレビュー、Excel書き出し |
| 設定 | 集計期間・年度名・前年度繰越金・月会費、カテゴリの追加/削除、バックアップ |

## 部費の管理（「部費」タブ）
部費は入力フォームではなく、専用のグリッドで管理します。
1. 「メンバー」に名前を追加
2. メンバー×月の表で、各セルを**タップ**してメニューから状態を選ぶ
   - ● 済（1,000円）／ ◯ 予定・未集金（1,000円）
   - ● お休み済（500円）／ ◯ お休み予定（500円）／ 対象外（空白）
3. 「済合計（実際に集まった額）」が収入の「部費」に自動反映。「予定合計」「未集金」も表示
- 金額は設定の「月会費」（既定1,000円）と、その半額（お休み月）で自動計算
- グリッドの月は「集計期間」に連動。Excelの部費内訳は「1,000円×Nヶ月 ＋ 500円×Mヶ月」形式で出力

## 使い始め（初回設定）
1. 「設定」タブを開く
2. **年度名**（例：2025年度）、**前年度繰越金**を入力して保存
3. 必要ならカテゴリを追加（既定で収入7種・支出10種が入っています）

## 毎回の使い方
1. 「入力」タブで収支を記録（支出/収入を切替 → カテゴリ・金額・摘要を入力 →「追加する」）
2. 「一覧」で残高を確認
3. 年度末は「集計」で内容を確認 →「Excelで書き出す」

## 会計報告プレビュー（印刷・PDF）
「集計」タブの「**会計報告プレビュー（印刷・PDF）**」ボタンで、提出フォーム風のレイアウト（収入／支出の表＋差引残高、区分は縦書き）を全画面表示します。
- 「**印刷／PDF保存**」ボタンでブラウザの印刷へ。プリンタ出力のほか、保存先で「PDF」を選べばPDF化できます
- 印刷時はアプリのメニュー等は自動で隠れ、報告書だけがきれいに出ます
- 実際の現金残高を入力していれば、差額（過不足）も併記されます

## バックアップと引き継ぎ（重要）
データは**使う端末の中**にしか保存されません。会計担当が毎年変わるチアフルでは、**Googleドライブの共有フォルダ**で歴代の記録を引き継ぐ運用がおすすめです。

### 共有フォルダのつくり方（一度だけ）
1. Googleドライブで **「チアフル会計」フォルダ**を作成
2. 右クリック→**共有**→歴代会計と現会計のGoogleアカウントを「編集者」で追加
3. 以後、フォルダのリンクをチームで引き継ぐだけで永続化

### 年度末の流れ
1. アプリの「設定」→「📥 **バックアップを書き出す（JSON）**」
2. ダウンロード時に**保存先で「Googleドライブ → チアフル会計フォルダ」**を選ぶ（iCloud Driveでも可）
3. 「集計」→「会計報告プレビュー」→「印刷／PDF保存」で**PDFも同フォルダに保存**

### 引き継ぎの流れ
1. 新会計はアプリを開き、「設定」→「📤 **前任のバックアップを読み込む（JSON）**」
2. 共有フォルダから最新のバックアップJSONを選択 → 過去のメンバー・カテゴリ・全データが取り込まれる
3. 「設定」→ 集計期間を新年度に変更、前年度繰越金を入力 → 引き継ぎ完了

### アプリ内の補助
- 期間の最終月・終了後は**設定タブ上部に黄色の警告**が出ます（バックアップ忘れ防止）
- **最終バックアップ日**も設定タブに表示

## ローカルでの起動（開発・確認用）
```
cd チアフル会計アプリ
python3 -m http.server 8778
# ブラウザで http://localhost:8778 を開く
```

## スマホで使えるようにする（公開）
オフライン機能（Service Worker）はURL公開が前提です。無料の静的ホスティングが簡単です。
- **GitHub Pages** または **Netlify** にこのフォルダを公開
- 公開URLをスマホで開き、ブラウザの共有メニューから「ホーム画面に追加」

※ 公開作業は別途サポートします。

## 技術メモ
- 構成: HTML / CSS / バニラ JS（ビルド不要）
- ストレージ: IndexedDB（取引・カテゴリ・設定・領収書画像）
- Excel: SheetJS（`lib/xlsx.full.min.js`、オフライン用にローカル同梱）
- オフライン: Service Worker（`sw.js`）＋ manifest（`manifest.json`）
